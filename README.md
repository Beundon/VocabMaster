# VocabMaster

A complete English vocabulary learning app with 9000 words across A2-C1 levels.

## Features
- User authentication (Sign up/Sign in/Log out)
- 9000 vocabulary words with definitions, examples, synonyms
- Practice modes: Flashcards, Quiz, Spelling, Review
- Progress tracking and statistics
- Export/Import data (Excel, CSV, JSON)
- Works offline as PWA

## Install as App
1. Open the live demo in Chrome/Safari
2. Tap "Add to Home Screen"
3. Works offline!
